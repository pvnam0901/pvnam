Thuộc tính id trong html

    id trong html giống như là số CMND của 1 người
    nó sẽ là duy nhất trong trang html và không được phép trùng giữa các dòng
    id phải viết liền và k có dấu, không có kí tự đặc biệt như @ # $...
    tốt nhất chỉ nên có chữ cái và các chữ số.

    class được sử dụng để gom nhóm giữa các thẻ html có điểm chung
    1 thẻ html có thể có nhiều hoặc 1 class khác nhau
    các class thì cần phân tách bằng dấu cách ví dụ:"mienbac top1 thudo"
    quy tắc viết tên class viết liền và không có dấu, không có kí tự đặc biệt như @#$%..
    - mục đích của thuộc tính class và id là để kết hợp với các ngôn ngữ như css và javascript
    để thực hiện các tác vụ trên trang web
    - trong css thì chọn class bằng dấu . ở đầu ví dụ .mienbac
    - trong css để chọn id sẽ viết #id ví dụ #hanoi
